    <h2>Technologies for intelligent and scalable glass optics manufacturing</h2>
    
    <div class="richtext">
    	<p>The WaferLevelOptics project aims to develop precision glass molding techniques for the replication of micro-optics at wafer level. It is foreseen that the molding of multiple micro-optics on one glass wafer, stacking these to optical systems, bonding them and then dicing them will become the norm for micro-optics manufacturing in future. 
<br/> 
      <br/> 
This project is funded by the European Comission within the 7th Framework (FP7).</p>
